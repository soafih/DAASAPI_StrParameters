DAAS Build API

Overview

DAAS build API is a REST webservice used to create applications, which access the data from various data-sources. The API accepts the application related information along with the Database connection details.


REST Endpoint

http://daasapi.10.135.4.49.xip.io/FIH/service/DAASAPI with method : POST


Sample Request

{
  "organization":"TESTORG",
  "space":"TESTSPACE",
  "applicationName":"TESTAPP",
  "query":"SELECT NAME FROM T1 WHERE ID='#ID#',
  "databaseInfo":
    {
    "databaseType":"ORACLE",
    "hostName": "HOSTNAME",
    "port": "PORT",
    "databaseName":"DBNAME",
    "schema":"LIBNAME",
    "user":"USERID",
   "password":"PWD"
    },
  "connectionAttr":
    {
    "maxActive":"50",
    "maxIdle":"30",
    "maxWait":"10000"
    }
}

Sample Response

{
  "response":
  {
    "result":"success",
    "logURL":"URL_TO_LOG",
    "app_ep":"SERVICE_ENDPOINT",
  }
}
